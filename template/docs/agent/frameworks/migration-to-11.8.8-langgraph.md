# LangGraph agent migration to 11.8.8

This guide covers migrating a LangGraph agent from the class-based layout (pre-11.8.8) to the new factory-based layout introduced in [af-component-agent#474](https://github.com/datarobot-community/af-component-agent/pull/474).

## Summary of changes

- `MyAgent` is no longer a hand-written class extending `LangGraphAgent`. It is generated by `datarobot_agent_class_from_langgraph()`.
- The `llm()` method is removed. LLM creation is decoupled from the agent and handled by `get_llm()` from `datarobot_genai.langgraph.llm`.
- The `Config` class import is removed. Configuration is handled internally by `get_llm()`.
- Agent nodes and the graph are defined inside a `graph_factory()` function instead of as class properties.
- The prompt template is defined at module level instead of as a class property.

## Migration steps

### 1. Update imports

**Before:**

```python
from datarobot_genai.langgraph.agent import LangGraphAgent
from langchain_litellm.chat_models import ChatLiteLLM

from agent.config import Config
```

**After:**

```python
from datarobot_genai.langgraph.agent import datarobot_agent_class_from_langgraph
from datarobot_genai.langgraph.llm import get_llm
```

Remove imports of `LangGraphAgent`, `ChatLiteLLM`, and `Config`.

### 2. Move `prompt_template` to module level

**Before:**

```python
class MyAgent(LangGraphAgent):
    @property
    def prompt_template(self) -> ChatPromptTemplate:
        return ChatPromptTemplate.from_messages([
            ("system", "You are a helpful assistant. Chat history: {chat_history}"),
            ("user", "The topic is {topic}."),
        ])
```

**After:**

```python
prompt_template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant. Chat history: {chat_history}"),
    ("user", "The topic is {topic}."),
])
```

### 3. Replace the class with a `graph_factory` function

Extract your `workflow` property and agent node properties into a standalone factory function. The function receives `llm`, `tools`, and `verbose` as arguments instead of reading them from `self`.

**Before:**

```python
class MyAgent(LangGraphAgent):
    @property
    def agent_planner(self) -> Any:
        return create_agent(
            self.llm(),
            tools=self.tools,
            system_prompt=make_system_prompt("You are a content planner. ..."),
            name="planner_agent",
        )

    @property
    def agent_writer(self) -> Any:
        return create_agent(
            self.llm(),
            tools=self.tools,
            system_prompt=make_system_prompt("You are a content writer. ..."),
            name="writer_agent",
        )

    @property
    def workflow(self) -> StateGraph[MessagesState]:
        langgraph_workflow = StateGraph(MessagesState)
        langgraph_workflow.add_node("planner_node", self.agent_planner)
        langgraph_workflow.add_node("writer_node", self.agent_writer)
        langgraph_workflow.add_edge(START, "planner_node")
        langgraph_workflow.add_edge("planner_node", "writer_node")
        langgraph_workflow.add_edge("writer_node", END)
        return langgraph_workflow
```

**After:**

```python
def graph_factory(
    llm: BaseChatModel, tools: list[BaseTool], verbose: bool = False
) -> StateGraph[MessagesState]:
    agent_planner = create_agent(
        llm,
        tools=tools,
        system_prompt=make_system_prompt("You are a content planner. ..."),
        name="planner_agent",
        debug=verbose,
    )

    agent_writer = create_agent(
        llm,
        tools=tools,
        system_prompt=make_system_prompt("You are a content writer. ..."),
        name="writer_agent",
        debug=verbose,
    )

    langgraph_workflow = StateGraph(MessagesState)
    langgraph_workflow.add_node("planner_node", agent_planner)
    langgraph_workflow.add_node("writer_node", agent_writer)
    langgraph_workflow.add_edge(START, "planner_node")
    langgraph_workflow.add_edge("planner_node", "writer_node")
    langgraph_workflow.add_edge("writer_node", END)
    return langgraph_workflow
```

Key differences:
- `self.llm()` &rarr; `llm` (passed as argument)
- `self.tools` &rarr; `tools` (passed as argument)
- `self.verbose` &rarr; `verbose` (passed as argument, forwarded as `debug=verbose`)

### 4. Generate `MyAgent` with the factory

Replace the entire `MyAgent` class with a single line:

```python
MyAgent = datarobot_agent_class_from_langgraph(graph_factory, prompt_template)
```

### 5. Delete the `__init__` and `llm()` methods

The entire `__init__` method (with `self.config`, `self.default_model`, `self._llm`, etc.) and the `llm()` method are no longer needed. Remove them.

### 6. Update `custompy_adaptor`

**Before:**

```python
async def custompy_adaptor(completion_create_params, ...):
    ...
    agent = MyAgent(
        model=completion_create_params.get("model"),
        verbose=...,
        timeout=...,
        forwarded_headers=...,
    )
```

**After:**

```python
_PLACEHOLDER_MODELS = frozenset({"unknown"})

async def custompy_adaptor(completion_create_params, ...):
    ...
    model_name = completion_create_params.get("model")
    agent = MyAgent(
        llm=get_llm(
            model_name=model_name if model_name not in _PLACEHOLDER_MODELS else None
        ),
        verbose=...,
        timeout=...,
        forwarded_headers=...,
    )
```

Key differences:
- `model=` parameter &rarr; `llm=get_llm(model_name=...)` parameter.
- `_PLACEHOLDER_MODELS` filters out the `"unknown"` model placeholder sent by DataRobot.

### 7. Update tests

Tests that mock `ChatLiteLLM`, test the `llm()` method, or instantiate `MyAgent` with `model=` need updating:

- Remove tests for `llm()`, `llm_with_identity_token`, etc.
- Replace `MyAgent(model=..., api_key=...)` with `MyAgent(llm=Mock(), ...)`.
- Test `graph_factory` directly: pass a mock LLM and tools, verify node names in the returned graph.
- Test `prompt_template` as a module-level object.
- Test `custompy_adaptor` with mock `get_llm` to verify placeholder model filtering.
